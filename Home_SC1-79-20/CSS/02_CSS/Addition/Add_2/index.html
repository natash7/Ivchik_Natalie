Блок навигации

  • Ширина 15em
  • Должен присутствать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано на данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствать цвет фона
Во всех "статях" сделайте отступ снизу от соседей равным 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным 2 em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри даной статьи для этой картинки.
Нижний блок занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
